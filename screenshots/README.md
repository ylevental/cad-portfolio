## Tips

- Use **white or dark background** for consistency
- Crop to the viewport — no need for full desktop screenshots
- **1920×1080** or similar is plenty for GitHub README embedding
- For OpenSCAD, press F6 (Render) before screenshotting for a clean solid view
- For FreeCAD, use View → Standard views → Isometric for a good angle
- For LibreCAD, use View → Zoom → Auto to fit the drawing
